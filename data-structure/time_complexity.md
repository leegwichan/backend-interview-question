# 시간 복잡도

### 시간 복잡도 & 공간 복잡도

- 시간 복잡도 : 입력값이 늘어날 때 처리 시간이 얼마나 가파르게 증가하는지를 나타내는 척도
  - 사용자/데이터가 급격하게 늘어났을 때, 처리 시간이 어떻게 바뀌는지 알기 위한 방법
- 공간 복잡도 : 입력값이 늘어날 때 메모리를 얼마나 추가로 사용하는지를 나타내는 척도
  - 사용할 수 있는 메모리는 한정되어 있으므로, 불필요하게 큰 배열을 잡거나 재귀 호출을 너무 깊게 하여 `Stack Overflow`나 `OOM(Out of Memory)`가 발생하지 않도록 관리해야한다.

### 자주 쓰이는 Big-O 표기법

| 표기법 | 이름 | 설명 (면접용 멘트) | 대표 예시 |
| --- | --- | --- | --- |
| $O(1)$ | 상수 시간 | "데이터가 아무리 많아도 한 번에 찾습니다." | Hash Map 조회 |
| $O(log n)$ | 로그 시간 | "데이터가 늘어날 때마다 처리 단계가 절반씩 줄어듭니다." | 이진 탐색 |
| $O(n)$ | 선형 시간 | "데이터가 늘어나는 만큼 정직하게 비례해서 시간이 늘어납니다." | for 문 (전체 탐색) |
| $O(nlogn)$ | 선형 로그 | "데이터를 효율적으로 정렬할 때 걸리는 시간입니다." | Quick Sort, Merge Sort |
| $O(n^2)$ | 이차 시간 | "데이터가 조금만 늘어도 시간이 제곱으로 늘어나 비효율적입니다." | 이중 for 문 |
